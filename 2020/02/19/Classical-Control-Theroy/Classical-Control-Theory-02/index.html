02 Closed Loop Control

发表于 2020-02-19

本节介绍开环控制和闭环控制。

控制系统

控制系统（Control System）能够修改系统未来的行为或状态，而且这种改变是具有目的性的，那就是使得系统行为或状态达到我们预期的理想状态。控制理论（Control Theroy）是数学的一个分支，以研究一种策略来选择一种合适的系统输入，使得系统输出如我们所愿。如果没有控制理论，系统设计者只能通过试错（trial and error）的方式来决定系统的行为，这种方式是十分低效的。

所有的控制系统，包含两部分：

  • 系统。能被控制的系统，通常被称为Plant。
  • 输入。即输入到或作用于Plant的输入。

开环控制

“ 系统 ”在“ 输入 ”的作用下会产生随时间变化的“ 输出 ”。这种系统被称为开环系统（open-loop system）,因为系统的“ 输入 ”不依赖于系统的“ 输出 ”。

开环控制例子

开环系统在现实生活中仍然被保留，一般用于系统的输入、输出行为被明确定义的场景。例如：

  • 洗碗机。洗碗机是Plant，即 被控系统，其目标是清洗碗碟。一旦用户设置了清洗时长，洗碗机就会持续清洗这么长时间，此处“清洗时长”就是被控系统的 输入。这种洗碗机并不关心碗碟是否清洗干净，而只是根据用户所设定的时长来工作。
  • 草地自动洒水装置。草地是 被控系统*，即Plant。该系统的 *输出 是土壤的水分含量，另外，用户设置的喷洒时长是系统的 输入。该系统只能完全执行所设置的工作时长，而不管土壤实际的水分含量到底有多少。
  • 匀速行驶的汽车。假设该汽车没有自动巡航控制系统（automatic cruise contorl）。汽车是 被控系统*，即Plant，系统的 *输出 是汽车行驶速度，系统的 输入 是汽车油门踏板的位置。初始时，汽车不断加速，直到施加在汽车上的牵引力和汽车所受到的汽车摩擦力相平衡，此时，汽车停止加速，并保持匀速行驶。当汽车遇到上坡或下坡时，如果不改变系统 输入，即调节油门踏板的位置，汽车就会在上坡时减速，在下坡时加速，因此就无法继续保持匀速了。

通过上述例子，可以看出，开环控制的缺点是：系统的 输入 无法补偿系统的变化。为了考虑系统的变化，可以根据系统的 输出 来改变系统的 输入 ，这种控制系统被称为闭环控制系统。

闭环控制

闭环控制系统的输入依赖于系统的输出，另外，闭环控制还被称为，“反馈控制”、“负反馈控制”或“自动控制”。虽然这些术语的含义有一些差异，但在本系列视频中不做区分。在闭环控制中，使用 传感器 测量系统的 输出*，并将传感器测量结果（即 *反馈 ）与 参考值 进行比较（通常这个 参考值 是系统的期望达到的理想状态）。然后将比较产生的 误差 输入到一个 控制器 中，该控制器将 误差 转化为系统（即Plant）的 输入*。顺便说一句，“负反馈控制”术语中的“负反馈”体现在 *参考值 和 反馈 在比较器那里进行了相减操作。整个闭环控制框图如下所示：

闭环控制例子

仍然用前面的例子来进行说明：

  • 洗碗机。 添加一个检测盘子是否清洗干净的 传感器*，而 *参考值 是由生产厂商或用户制定的一种衡量盘子清洁度的清洁等级，这个参考值 将与传感器测量的实际清洁等级进行比较，从而产生一个误差*，然后将这个 *误差 输入到 控制器 中，这个控制器通过监测这个 误差 来监测系统的状态。当 误差 小于一定阈值时，说明系统的当前状态与系统的目标状态足够接近，此时就可以关闭洗碗机。
  • 草地自动洒水装置。添加一个传感器来测量土地中的湿度，此时的Plant包括草和土地。参考值 可以是期望的土地湿度。参考值 与 反馈相减得到 误差，并将误差输入到控制器中，通过控制器来调节洒水装置运行时长。
  • 匀速行驶的汽车。具有自动巡航控制系统的汽车调速过程大致是，使用里程计（speedometer）作为测量车速的 传感器 ，参考值 是自动巡航控制系统所设定的理想车速。假设初始时汽车在平地上匀速运动，此时里程计测量的车速为100MPH。假设汽车遇到一个上坡，里程计测量的车速为90MPH，故 参考值 减去 测量值 为+10。此时控制器会让汽车加速。假设汽车遇到一个下坡，里程计测量的车速为110MPH，故 参考值 减去 测量值 为-10。此时控制器会让汽车减速。

反馈控制系统的好处是其能够自动对Plant的变化做出反应，并使得误差项逐渐趋于0。
