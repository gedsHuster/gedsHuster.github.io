在Markdown中插入图片

发表于 2020-02-21

本文介绍如何将七牛云作为图床在Markdown中插入图片。

在本地显示图片

Markdown提供了两中插入图片的方式（具体见Image syntax）：

  • inline方式，如：

    • ![Alt text](/path/to/img.jpg)

    • ![Alt text](/path/to/img.jpg "Optional title")

      即，感叹号！ + 方括号[] + 圆括号()，[]中可填写描述照片的信息，()中填写URL或图片在本地的保存路径。

  • reference方式，如：

    • ![Alt text][id]

    • [id]: url/to/image "Optional title attribute"

      其中，“id”是图片引用的名字，且图片引用[id]由图片链接url/to/image来定义。

在静态网页上显示图片

理论上，使用以上两种方式是可以将图片加入到Markdown文档中，然而实际上图片并不能正常显示，不管是将.md文件保存为本地的pdf或html文件，或将其部署成静态网页，都不能显示图片。为了解决这个问题，从网上找到一种解决方案，即使用图床作为存储图片的服务器，并生成外部链接，然后将该链接按照inline法或reference法插入到Markdown中即可。
